WNS allows instead of long and complex addresses, like, '3PEaRctfe4i7xxYhDjj', to set a simple intuitive name, such as “sasha.waves”, thereby providing an incredible user experience.
Want to make a transfer? Just ask a friend for his WNS address, "myfriend.wallet.waves", and translate any WNS compatible wallet.
WNS is a complete set of tools for early integration into dapp applications and wallets.

Secure.

WNS is a direct alternative to the familiar Domain Name System and the path to Web 3.0, a truly decentralized Internet.
WNS is built on the smart contracts of the Waves network, written in RIDE, providing a completely decentralized approach to managing and registering names. This is the way to Web 3.0.

Truly distributed.

WNS is completely distributed, ranging from infrastructure to management. Anyone can register or buy a domain name in allowed zones through auctions on the Waves blockchain.
WNS is managed by a decentralized autonomous organization (DAO) providing bureaucracy-free platform infrastructure management.
